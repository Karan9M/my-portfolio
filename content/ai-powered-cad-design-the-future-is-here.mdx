## What is Adam, and How Does it Work?
Adam is built on two fundamental beliefs: AI will be the primary medium for creating mechanical designs, and the best paradigm for CAD generation is to generate CAD as code (text -> code -> CAD). This means that instead of manually designing CAD models, you can use AI to generate them automatically. The team at Adam has built an open-source Text to CAD platform called CADAM, which is a React app with a Supabase backend for auth, database, and file storage. Think of it like AI TinkerCAD - it's a powerful tool that can generate parametric CAD models from text prompts.

### Key Features of CADAM
1. **Generates parametric CAD models**: CADAM can generate complex CAD models from simple text prompts, which can be customized and fine-tuned to meet specific design requirements.
2. **Uses AI to optimize designs**: The AI engine can optimize CAD designs for performance, cost, and other factors, which can help reduce the time and effort required to create and iterate on designs.
3. **Open-source and customizable**: CADAM is open-source, which means that developers can customize and extend the platform to meet their specific needs.
