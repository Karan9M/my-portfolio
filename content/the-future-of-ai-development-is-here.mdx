## What's Changing
The Agents SDK is a powerful tool for developers, allowing us to build complex AI agents that can interact with various tools and files. But, as with any powerful tool, security and stability are top concerns. That's where the latest update comes in – with native sandbox execution and a model-native harness, we can now build secure, long-running agents with ease.

## Diving Deeper
The update includes two major features: native sandbox execution and a model-native harness. The native sandbox execution allows agents to run in a secure, isolated environment, which is a game-changer for developers who want to build complex AI agents without worrying about security risks. The model-native harness, on the other hand, provides a standardized way to interact with AI models, making it easier to build and integrate agents into various applications.
